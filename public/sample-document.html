Sample PDF Document

Introduction

This is a sample PDF document for testing the React PDF viewer component.

Features

  • Page navigation (previous/next)
  • Zoom controls (in/out/reset)
  • Responsive design
  • Error handling
  • Loading states

Instructions

To use this PDF viewer:

  1. Replace this HTML file with a real PDF file named 'sample.pdf'
  2. Use the navigation buttons to move between pages
  3. Use zoom controls to adjust the view
  4. The component is fully responsive and accessible

Note

To create a PDF from this HTML file:

1. Open this file in a browser

2. Press Ctrl+P (or Cmd+P on Mac)

3. Select "Save as PDF" as the destination

4. Save as "sample.pdf" in the public directory
